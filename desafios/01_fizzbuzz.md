# Desafio 01: FizzBuzz

## Descrição
Escreva uma função que recebe um número inteiro `n` e retorna uma lista de strings de 1 até `n`, onde:
- Se o número é divisível por 3, substitua por "Fizz"
- Se o número é divisível por 5, substitua por "Buzz"
- Se o número é divisível por ambos 3 e 5, substitua por "FizzBuzz"
- Caso contrário, retorne o número como string

## Exemplo
```python
fizzbuzz(15)
# Retorna: ['1', '2', 'Fizz', '4', 'Buzz', 'Fizz', '7', '8', 'Fizz', 'Buzz', '11', 'Fizz', '13', '14', 'FizzBuzz']
```

## Assinatura da Função
```python
def fizzbuzz(n: int) -> list[str]:
    pass
```
