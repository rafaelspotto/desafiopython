# Desafio 08: Two Sum

## Descrição

## Exemplo
```python
two_sum([2, 7, 11, 15], 9)
# Retorna: [0, 1]  # porque nums[0] + nums[1] = 2 + 7 = 9

two_sum([3, 2, 4], 6)
# Retorna: [1, 2]  # porque nums[1] + nums[2] = 2 + 4 = 6

two_sum([3, 3], 6)
# Retorna: [0, 1]
```
